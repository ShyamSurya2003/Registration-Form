# Registration Form with Validation

## 📌 Project Description

This project is a simple **Registration Form with real-time input validation** built using **HTML, CSS, and JavaScript**.
It validates user inputs such as **Name, Email, and Password** before allowing the form to be submitted.

The form provides instant feedback using color indicators and error messages, improving the user experience.

---

## 🚀 Features

* Real-time form validation
* Email format validation
* Password length validation
* Disabled submit button until all fields are valid
* Visual feedback (green = valid, red = error)
* Responsive and modern UI
* Reset form after successful submission

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the form
* **CSS3** – Styling and layout
* **JavaScript (Vanilla JS)** – Form validation and interactivity

## ⚙️ How to Run the Project

1. Download or clone the project.
2. Open the project folder.
3. Double click **index.html**.
4. The form will open in your web browser.

No server is required.

---

## 📖 How It Works

1. User enters **Name, Email, and Password**.
2. JavaScript validates each input.
3. If input is invalid:

   * Error message is shown
   * Input border turns **red**.
4. If input is valid:

   * Border turns **green**.
5. Submit button becomes enabled only when all fields are valid.

---

## 📸 Example Validation Rules

| Field    | Rule                           |
| -------- | ------------------------------ |
| Name     | Cannot be empty                |
| Email    | Must follow valid email format |
| Password | Minimum 6 characters           |
